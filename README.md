# AI Native Dev Starter

AI 코딩 에이전트를 보조 도구가 아니라 개발 워크플로의 기본 단위로 쓰기 위한 문서 스타터팩입니다.

이 레포는 팀이나 개인이 바로 복사해서 쓸 수 있는 Markdown 문서, 체크리스트, 프롬프트 템플릿을 제공합니다.

## Who This Is For

- AI 코딩 에이전트로 기능 구현, 리팩터링, 테스트, 리뷰를 반복하려는 개발자
- Cursor, Codex, Claude Code, Copilot Coding Agent 같은 도구를 팀 워크플로에 붙이려는 팀
- "바이브 코딩"을 프로덕션 개발 방식으로 끌어올리고 싶은 사람

## Core Idea

AI 네이티브 개발은 "AI에게 코드를 맡기는 것"이 아니라, 사람이 문제 정의와 검증 기준을 설계하고 에이전트가 반복 작업과 탐색을 빠르게 수행하도록 만드는 개발 방식입니다.
